# README

## 日本語

[Architect's Log](https://blog.jhashimoto.net/) のブログ記事を管理するリポジトリです。

[hatena/Hatena-Blog-Workflows-Boilerplate](https://github.com/hatena/Hatena-Blog-Workflows-Boilerplate) をテンプレートとして作成しました。GitHub Actions を使い、はてなブログの記事とリポジトリ内の Markdown ファイルを同期しています。

## English

This repository manages blog posts for [Architect's Log](https://blog.jhashimoto.net/).

It was created using [hatena/Hatena-Blog-Workflows-Boilerplate](https://github.com/hatena/Hatena-Blog-Workflows-Boilerplate) as a template. GitHub Actions are used to sync blog posts with Markdown files in this repository.

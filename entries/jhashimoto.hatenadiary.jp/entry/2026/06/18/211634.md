## PostgreSQLのバージョンポリシー

https://www.postgresql.org/support/versioning/

> The PostgreSQL Global Development Group supports a major version for 5 years after its initial release. After this, a final minor version will be released and the software will then be unsupported (end-of-life).
意訳。

- メジャーバージョンの初回リリース後、5 年間サポートが継続される
- 最終的なマイナーバージョンのリリースをもって、そのメジャーバージョンは EOL となる

## PostgreSQLのバージョンを調べる

サーバー。

```sh
psql -U postgres -c "SHOW server_version;"
# server_version
#----------------
# 15.15
```

クライアント。

```sh
psql --version
# psql (PostgreSQL) 15.15
```

## パーティションのファイルシステムを知りたい

`df`に`-t` (Type) オプションを付ければよい。

```sh
df -h
# ファイルシス                        サイズ  使用  残り 使用% マウント位置
# ...
# /dev/nvme0n1p1                        xxxG   xxG   xxG   xx% /
```

```sh
df -hT
# ファイルシス                        タイプ   サイズ  使用  残り 使用% マウント位置
# ...
# /dev/nvme0n1p1                      xfs        xxxG   xxG   xxG   xx% /
```

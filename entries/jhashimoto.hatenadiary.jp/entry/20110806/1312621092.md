**公式

階乗の記号を用いて、二項定数は以下のように書ける。

>||
nＣr = n! / r!(n - r)!
||<

>>
これは、相違なるn個のものから、r個を取る組合せ(combnation)の総数を示す。文字ＣはCombinationの頭文字である。
<<

**公式の確認
nとrに具体的な数を入れて、確認してみる。

>||
1Ｃ1 = 1! / 1!0!
= 1

2Ｃ1 = 2! / 1!1!
= 2

3Ｃ2 = 3! / 2!1!
= 6 / 2
= 3
||<
確かにその通りになっている。

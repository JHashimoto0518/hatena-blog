単純な属性セレクタは、指定された属性をもつ要素にスタイルを適用します。
**使用例
>|html|
<!DOCTYPE html>
<html lang="ja">
<head>
<title>Hello! CSS</title>
<meta charset="UTF-8">
<style>
[class] {   /* class属性をもつ要素 */
    color:  red;
}
</style>
</head>
<body>
<p class="notification">第１段落です。</p>
<p class="infomation">第２段落です。</p>
<p>第３段落です。</p>
</body>
</html>

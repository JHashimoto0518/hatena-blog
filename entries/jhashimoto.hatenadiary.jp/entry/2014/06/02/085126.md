<h4>グリッドの問題</h4>
<ul>
<li><span style="line-height: 1.5;">作ってみると要素が縦長だったりしてきれいに見えないことがある</span></li>
</ul>
<h4>そこでmasonry</h4>
<ul>
<li><span style="line-height: 1.5;">グリッド系のデザインでスペースを有効活用して配置してくれる</span></li>
</ul>
<h4>masonry環境の指定</h4>
<ul>
<li><span style="line-height: 1.5;">jQueryを読み込む</span></li>
<li><span style="line-height: 1.5;">masonry.pkgd.min.jsを読み込む</span></li>
</ul>
<h4>masonry環境の指定２</h4>
<ul>
<li><span style="line-height: 1.5;">boxクラスが指定された要素にmasonryを適用する</span></li>
</ul>
<pre>$(function() {
	$('#container').masonry({
		itemSelector:'.box',
		columnWidth:200
	});
});</pre>
<h4>masonriyには豊富なオプションがある</h4>
<h4>注意事項</h4>
<ul>
<li><span style="line-height: 1.5;">画像の読み込みのタイミングが遅かったりすると、レイアウトが崩れてしまう時がある</span></li>
<li><span style="line-height: 1.5;">その場合は、画像をローディングした後、masonryを実行する必要がある</span></li>
</ul>
<h4>imagesLoadedプラグイン</h4>
<p><a href="https://github.com/desandro/imagesloaded" target="_blank"><img class="alignleft" src="http://capture.heartrails.com/150x130/shadow?https://github.com/desandro/imagesloaded" alt="" width="150" height="130" align="left" border="0" /></a><a style="color: #0070c5;" href="https://github.com/desandro/imagesloaded" target="_blank">desandro/imagesloaded · GitHub</a><a href="http://b.hatena.ne.jp/entry/https://github.com/desandro/imagesloaded" target="_blank"><img src="http://b.hatena.ne.jp/entry/image/https://github.com/desandro/imagesloaded" alt="" border="0" /></a></p>
<div style="clear: both;"> </div>
<ul>
<li><span style="line-height: 1.5;">imagesLoadedでローディングが完了した後にmasonryの処理を行う</span></li>
</ul>
<pre>$(function() {
	$('#container').imagesLoaded(function() {
		$('#container').masonry({
			itrmSelector:'.box',
			columnWidth:200
		});
	});
});</pre>
<h4>masonryを使って作ったもののデモ</h4>
<ul>
<li><span style="line-height: 1.5;">画像で作成した文字をバラバラに分割しておく</span></li>
<ul>
<li><span style="line-height: 1.5;">PCサイズだと読めない</span></li>
<li><span style="line-height: 1.5;">スマフォサイズにすると画像が整列されて文字が読めるようになる</span></li>
</ul>
</ul>

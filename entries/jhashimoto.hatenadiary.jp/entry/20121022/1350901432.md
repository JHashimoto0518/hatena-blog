要素セレクタには、HTML要素を指定します。
**使用例
>|html|
<!DOCTYPE html>
<html lang="ja">
<head>
<title>Hello! CSS</title>
<meta charset="UTF-8">
<style>
h2 {
    color:	red;
}
</style>
</head>
<body>
<h1>大見出し</h1>
<h2>中見出し</h2>
<h3>子見出し</h3>
</body>
</html>
